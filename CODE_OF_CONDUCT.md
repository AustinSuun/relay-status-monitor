# 行为准则

## 我们的承诺

为了营造开放、友善和安全的社区，我们承诺：无论年龄、身体或心理状况、种族、性别认同与表达、性取向、经验水平、国籍、外貌、宗教或其他个人特征如何，都以尊重和善意对待参与者。

## 期待的行为

- 使用包容、专业且建设性的语言。
- 尊重不同的观点、经验和边界。
- 接受善意反馈，关注问题而非针对个人。
- 在公开讨论中保护凭证、个人数据和其他敏感信息。

## 不可接受的行为

- 骚扰、歧视、威胁、人身攻击或故意暴露他人隐私。
- 发布未经脱敏的生产数据、凭证或个人信息。
- 其他在专业社区中明显不适当或有害的行为。

## 执行

可通过 GitHub 仓库的私密安全报告渠道联系维护者。维护者会公平调查并采取必要措施，包括提醒、删除内容、暂时限制参与或永久移除参与权限。

本准则参考 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)。
